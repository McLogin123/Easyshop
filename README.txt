Easyshop BD

Requisitos:
- Python, Flask, pymongo

Para instalar flask y pymongo:
- pip install Flask pymongo (en una terminal o en la consola de comandos)

Para que el programa empieze a funcionar siga estos pasos:
- Abrir una consola de comandos
- En la consola poner la ruta que tiene la aplicacion (ej. cd C:\Users\user\Documents\Easyshop)
- Ejecutar la app con "Python app.py"
- Copiar y pegar la IP dada en el navegador
